# Помічник банерів

Локальний інструмент для пакетної заміни банерів усередині батьківських скриншотів.

Інструмент не використовує AI. Він відкриває батьківське зображення, знаходить старий
зелений вертикальний банерний слот, вставляє новий банер і зберігає результат.

## Основний сценарій

За замовчуванням React-інтерфейс налаштований на простий щоденний кейс:

- будь-яка кількість батьківських зображень;
- відповідна кількість маленьких банерів;
- 1-й банер вставляється в 1-й скриншот, 2-й у 2-й, і так далі;
- на виході виходять готові зображення.

## React-Інтерфейс

```bash
npm install
npm run dev
```

Відкрийте `http://127.0.0.1:5173/`.

Інтерфейс дозволяє:

- вибрати окремі файли або цілу папку з батьківськими скриншотами;
- вибрати окремі файли або цілу папку з новими банерами;
- працювати у режимі `По парах` або `Усі комбінації`;
- використовувати автопошук зеленого банерного слота або ручні координати;
- вставляти банер у слот, з полями або 1:1 без зміни розміру;
- завантажити один результат або всі результати ZIP-архівом.

## Python-Режим

```bash
python3 -m pip install -r requirements.txt
mkdir -p parents banners results
```

Покладіть скриншоти у `parents/`, а JPG/PNG банери у `banners/`.

Запуск:

```bash
python3 banner_helper.py --parents parents --banners banners --output results --pairing zip
```

## Опції

`--pairing zip`

Працює по парах: `parent 1 + banner 1`, `parent 2 + banner 2`.

`--pairing all`

Створює всі комбінації. Наприклад, 20 скриншотів і 5 банерів дадуть 100 результатів.

`--mode fit`

Основний режим. Банер масштабується точно у знайдений слот.

`--mode contain`

Банер масштабується без обрізання, можуть з'явитися поля.

`--mode exact`

Банер вставляється 1:1 без зміни розміру.

## Ручні Координати

Якщо слот не потрібно шукати автоматично:

```bash
python3 banner_helper.py --parents parents --banners banners --output results --mode fit --x 638 --y 401 --width 156 --height 304
```

## CSV-Координати

CSV-файл:

```csv
parent,x,y,width,height
screen1.png,638,401,156,304
screen2.png,795,436,156,304
```

Запуск:

```bash
python3 banner_helper.py --parents parents --banners banners --output results --coords coords.csv
```
